结构命名 
•	Lens_Aperture (镜头孔径)：中心最深处的黑色光学区域（红色框内部）。缺陷特点：一般为缺陷极小的点伤。
•	Lens_Housing (镜头座主体)：由原先的“磨砂支架”与“内衬环”合并而成（红色框外部）。这是指从中心孔径边缘一直延伸到四个定位缺口的整个灰色结构。缺陷特点：划痕、尘埃、脏污；一般面积较大

### 镜头模组数据特性分析 

本研究根据镜头模组的物理结构与功能属性，将其空间域 $\Omega$ 划分为两个具有显著统计差异的子区域：**镜头孔径区 (Lens_Aperture)** 与 **镜头座主体区 (Lens_Housing)**。

#### 1. 镜头孔径区 (Lens_Aperture, $\Omega_{ap}$)

* **几何定义**：设定图像中心为原点，整图半径为 $R$。该区域定义为以原点为中心的圆形区域，其径向范围满足 $0 \le r < 0.25R$。在极坐标系下表示为 $\Omega_{ap} = \{(r, \theta) \mid 0 \le r < 0.25R, 0 \le \theta < 2\pi\}$。

#### 2. 空间背景统计特征

* **高纯净度与低信息熵**：此区域对应镜头的成像核心光学区, 表现为均匀、深邃的黑色光学吸光背景。从统计学视角看，其像素灰度分布高度集中在低亮度区间（$Pixel \in [0, 20]$），背景方差极小。
* **高信噪比 (SNR)**：由于排除了磨砂支架（Housing）的复杂纹理干扰，特征空间背景极其“纯净”。任何微小的亮度突变在空间频域上均表现为显著的非平稳脉冲信号。

#### 3. 典型缺陷形态学分类

本研究根据物理成因与视觉拓扑特征，将孔径区缺陷归纳为以下三类：

* **微米级点状缺陷 (Point Defects)**：
* **特征**：表现为高频、极小尺寸的亮斑，往往仅占据 $10 \times 10$ 至 $15 \times 15$ 像素区域。物理成因：多为封装过程中的细微粉尘、感光层物理坑洞或镀膜层局部的点状脱落。
* **挑战**：属于典型的“极小目标检测”问题，要求检测模型在多层特征融合过程中必须具备极高的空间分辨率保持能力，防止细微特征在下采样过程中被平滑。

* **镜面雾状污渍 (Smudge/Mist)**：
* **特征**：具有模糊的边缘边界与低对比度（Low-contrast）。在特征空间中表现为**低频偏移信号**，极易与镜头边缘的自然光影渐变（Vignetting）混淆。

* **镜片破损与崩边 (Fracture/Chipping)**：
* **特征**：表现为大面积、形状极不规则的白色或亮灰色斑块。它通常具有锐利的边缘和内部复杂的纹理（由于材质碎裂导致的多面反射）。物理成因：多为镜片安装时的机械硬碰撞、光学玻璃边缘的崩裂，或是内层消光涂层的成片脱落。  


#### 2. 镜头座主体区 (Lens_Housing, $\Omega_{ho}$)

#### 1. 空间几何定义

该区域定义为从孔径边缘向外延伸至模组物理边界的**环带区域 (Annular Region)**。其径向范围满足 $0.25R \le r < R$。在极坐标系下表示为：


$$\Omega_{ho} = \{(r, \theta) \mid 0.25R \le r < R, 0 \le \theta < 2\pi\}$$

此区域由支架与内衬环组成。在实际算法处理中，通常需利用掩码（Mask）剔除边缘处四个不规则的定位缺口，以获取统计意义上相对平稳的纹理流形。

#### 2. 空间背景统计特征

* **高频随机纹理背景 (High-frequency Texture)**：镜头座表面具有典型的**工业喷砂/磨砂工艺**特征。在特征空间中，表现为密集的、微观尺度的明暗交替，形成了极强的**各向同性随机噪声背景**。
* **纹理掩蔽效应 (Texture Masking Effect)**：这种高频背景会对判别器产生显著干扰，使得缺陷信号极易被淹没在正常的“颗粒感”中。使其难以区分“正常的颗粒感”与“异常的划痕/异物”。

#### 3. 典型缺陷形态学分类

根据装配工艺与损伤机理，该区域的缺陷主要分为以下两类：

* **结构性异常 (Macro-defects)**：
* **特征**：包括**长条形划痕 (Scratches)**、**大面积污渍 (Smudges)** 及 **玻璃碎屑 (Chips)**。
* **表现**：此类缺陷通常表现为**低频、大尺寸且具有显著方向性**的纹理冲突。在特征映射图中，它们会破坏磨砂表面的空间连续性，产生较强的局部残差响应。

* **低对比度近分布异常 (Low-contrast Anomalies)**：
* **特征**：包括**微小粉尘 (Dust)**、**脱漆点**及**边缘压伤 (Edge Dents)**。
* **表现**：点状异物在颜色空间（RGB）或特征深度上仅有微弱偏移；而边缘压伤表现为不规则的凹陷或纹理断裂，源于装配时的机械应力导致的**塑性变形**。
* **难点**：此类缺陷在尺度上与背景磨砂颗粒高度相近，属于**近分布异常（Near-distribution Anomalies）**。传统的 GLASS 模型在此类缺陷上极易产生漏检，必须依靠更精细的空间一致性约束（如代价滤波）进行细化判别。






图像特征与质量
•	分辨率与格式： 文件名为 .bmp 格式，分辨率为 1920x1920。BMP 是无损格式，保留了丰富的细节，适合工业检测。
•	色彩： 图像基本呈中性灰色调，只有镜头中心有微弱的彩色（光学镀膜引起）。
•	光照： 采用了非常均匀的环形光（Ring Light）或同轴光照明，阴影极小，能够清晰地勾勒出零件的轮廓和表面细微的颗粒感。

数据特点（研究对象）
•	检测对象：工业级镜头模组表面（Lens Surface）。
•	图像规格：原始分辨率为 1920*1920 的高分辨率 BMP 图像，具有明显的圆形几何对称结构和精密光学表面特征。
•	核心挑战：缺陷通常为极细微的划痕、尘埃或镀膜不均，这些缺陷在特征空间中与正常背景高度相似（Near-distribution Anomalies），且高分辨率图像对显存和计算效率提出了极高要求。
